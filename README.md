# Maint AI Web

Maint AI Web is a React + Vite based web application for managing and monitoring maintenance-related tasks using an AI-powered interface. The project is designed to provide a fast, responsive, and user-friendly experience for users who want to organize maintenance activities efficiently.

## Features

* Modern React-based user interface
* Fast development and build process with Vite
* AI-focused maintenance management interface
* Responsive design for different screen sizes
* Clean and scalable project structure
* ESLint support for better code quality

## Technologies Used

* React
* Vite
* JavaScript
* HTML
* CSS
* ESLint

## Project Setup

### 1. Clone the repository

```bash
git clone <your-repository-url>
```

### 2. Go to the project folder

```bash
cd maint-ai-web
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will usually run at:

```bash
http://localhost:5173
```

## Available Scripts

### Run development server

### Create production build

## Usage

After starting the development server, open the local URL in your browser. You can then use the web interface to explore and manage maintenance-related features.

## Build

To create a production-ready build, run:

```bash
npm run build
```

The build files will be generated inside the `dist` folder.
